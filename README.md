# Site A'Bonbonne

## Rodar localmente

```bash
make preview
```

Ou com outra porta:

```bash
make preview PORT=8080
```

## Arquitetura de páginas
- `index.html` — home com storytelling e conversão
- `eventos.html` — serviços por tipo de evento
- `cardapio.html` — builder com seleção visual e faixa de preço
- `espaco.html` — galeria do espaço
- `orcamento.html` — funil de lead com progresso
- `sobre.html` — posicionamento da marca
- `contato.html` — canais de atendimento
